# Roblox APK

Đặt file `roblox.apk` ở đây trước khi chạy `./install-roblox.sh`.

## Cách tải Roblox APK

1. **Từ APKMirror:** https://www.apkmirror.com/apk/roblox-corporation/roblox/
2. **Từ Roblox official:** https://www.roblox.com/mobile (Google Play)

⚠️ Không phân phối Roblox APK qua repo này (vi phạm Roblox ToS).
